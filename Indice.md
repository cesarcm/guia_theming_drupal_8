3. Twig basico.
	- Imprimir variables.
	- Set variables
	- if 
	- Twig | filters
	- Acceso a métodos en objetos con twig.

4. Instalacion de devel module.
	- Uso de kint para debug de las variables del sitio

5. Sobreescribir un template de twig
	- Modificar el markup 

6. Hook Preprocess
